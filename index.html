Welcome to Kenobi API

This library carry out verification of positioning within a determined area to check if the point of positioning is within that distance informed.

Like This :

In the case shown above you can check if the red pin google maps is contained in the area represented from point X, a radius of 4 km around the point X...

Example: 

Util util = new Util();

//point you want to check if it is within the region
Ponto pontoOrigem = new Ponto(-22,4837593,-43,4384765753); 

// this is center of region
Ponto x = new Ponto(-22,847359479,-43,483759387);  

// Raio of circle ==4km
boolean insideRegion = util.isNearPoints(pontoOrigem, x, 4); 
System.out.println(insideRegion);

for Java Projects : kenobi.jar file

for Android : library kenoby importing to your project

This is a good way to save your battery, when you use the google maps to check it, your battery dies very fast
